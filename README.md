# Shelf Technical Showcase

Small public-facing examples that show the technical direction of Shelf without exposing the full application logic.

## Included

- `text_analysis_basic.py` — basic text statistics and simple level estimation
- `difficulty_scoring_basic.py` — simple adaptive review difficulty idea
- `morphology_basic.py` — minimal word normalization examples
- `ocr_cleanup_basic.py` — simple OCR text cleanup utilities

These files are intentionally lightweight and are meant to demonstrate project structure, not the full production implementation.
