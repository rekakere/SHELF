# Shelf

Shelf is a reading-centered language learning platform designed to help learners acquire vocabulary and grammar through real texts rather than isolated flashcards.

The project combines reading, dictionary lookup, grammar discovery, adaptive review, and language analysis into a single workflow.

## Core Idea

Most language-learning tools separate reading, grammar, vocabulary, and review.

Shelf attempts to unify them into a single environment where language is learned through interaction with authentic texts.

Instead of memorizing isolated words, learners build knowledge from context, patterns, and repeated exposure.
