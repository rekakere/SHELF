# SHELF
Language Learning APP (text-based reading and vocabulary storage)

--> built around reading, discovery, and memory.

Instead of studying vocabulary lists, learners acquire words,
grammar patterns, and constructions directly from texts.

Built with:

• React + TypeScript
• FastAPI + Python
• spaCy NLP
• CEFR difficulty estimation
• Adaptive review system
